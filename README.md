**GhostQuery AI** es una herramienta de alta precisión diseñada para optimizar el flujo de trabajo con Inteligencia Artificial.
El usuario solo debe escribir una tarea o seleccionar una categoría, y la aplicación genera al instante una recomendación de IA, las habilidades necesarias y un **prompt avanzado** listo para copiar y utilizar.

---

Puedes utilizar la aplicación directamente desde tu navegador sin instalar nada:

## 🚀 Funcionalidades principales

* 🤖 Recomienda la IA más adecuada según la tarea del usuario.
* 🧠 Genera skills o conocimientos necesarios para realizar la tarea.
* ✍️ Crea prompts avanzados listos para copiar y usar.
* 📂 Organiza categorías por áreas como tecnología, creatividad, negocios, educación y seguridad.
* ⚙️ Incluye modos de respuesta: rápido, experto, técnico y creativo.
* 🧹 Permite limpiar el chat actual de forma rápida.
* 🎨 Presenta una interfaz oscura, moderna y minimalista.

## 🛠️ Tecnologías utilizadas

* **Frontend:** Streamlit con diseño personalizado.
* **IA:** LangChain + Google Gemini Generative AI API.
* **Lenguaje:** Python.
* **Gestión de variables:** python-dotenv.
* **Control de versiones:** Git + GitHub.
* **Deploy:** Streamlit Community Cloud.

---

## 🧠 ¿Cómo funciona?

1. El usuario selecciona una categoría o escribe una tarea personalizada.
2. GhostQuery AI analiza la solicitud.
3. La app recomienda la IA más adecuada.
4. Genera skills necesarios para realizar la tarea.
5. Crea un flujo de trabajo sugerido.
6. Produce un prompt avanzado listo para copiar y usar.

---

## 🎯 Objetivo del proyecto

El objetivo de **GhostQuery AI** es facilitar el uso práctico de herramientas de inteligencia artificial, ayudando a estudiantes, desarrolladores, creadores de contenido, emprendedores y profesionales a elegir la IA correcta para cada necesidad.
